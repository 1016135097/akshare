### 电影票房-实时

接口: movie_board

目标地址: https://maoyan.com/board/1

描述: 获取最新电影的实时票房数据

限量: 将昨日国内热映的影片, 按照昨日票房从高到低排列, 每天上午10点更新

输入参数

| 名称   | 类型 | 必选 | 描述                                                                              |
| -------- | ---- | ---- | --- |
| - | -  | -    |  -|


输出参数

| 名称          | 类型 | 默认显示 | 描述           |
| --------------- | ----- | -------- | ---------------- |
| -     | float   | -        | json返回所有推送数据, 逐步做处理  |  


接口示例

```python
import akshare as ak
movie_df = ak.movie_board()
print(movie_df)
```
