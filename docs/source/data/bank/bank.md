## [AkShare](https://github.com/jindaxiang/akshare) 银保监数据

### 银保监分局本级行政处罚

接口: bank_fjcf(需要安装 **Nodejs**)

目标地址: http://www.cbirc.gov.cn/cn/list/9103/910305/ybjfjcf/1.html

描述: 获取银保监分局本级行政处罚

限量: 单次返回银保监分局本级行政处罚中的表格数据

输入参数

| 名称   | 类型 | 必选 | 描述                                                                              |
| -------- | ---- | ---- | --- |
| page | int  | Y    |  page=5, 获取5页数据|

输出参数

获取银保监分局本级行政处罚

| 名称          | 类型 | 默认显示 | 描述           |
| --------------- | ----- | -------- | ---------------- |
| 行政处罚决定书文号      | -   | -        | -  |  
| 姓名      | -   | -        | -  |  
| 单位      | -   | -        | -  |  
| 名称      | -   | -        | -  |  
| 主要负责人姓名      | -   | -        | -  |  
| 主要违法违规事实（案由）      | -   | -        | -  |  
| 行政处罚依据      | -   | -        | -  |  
| 行政处罚决定      | -   | -        | -  |  
| 作出处罚决定的机关名称      | -   | -        | -  |  
| 作出处罚决定的日期      | -   | -        | -  |  

			
接口示例

```python
import akshare as ak
df = ak.bank_fjcf(page=3)
print(df)
```
