AkShare 能源数据¶

碳排放权¶

接口: energy_carbon

目标地址: https://www.bjets.com.cn/article/jyxx/

描述: 获取北京市碳排放权电子交易平台-北京市碳排放权公开交易行情

限量: 全部历史数据

输入参数

名称 类型 必选 描述
- - - -

输出参数

名称 类型 默认显示 描述
日期 str -
成交量(吨) float - 注意单位
成交均价(元/吨) float - 注意单位
成交额(元) str - 注意单位

接口示例

import akshare as ak
energy_carbon_df = ak.energy_carbon()
print(energy_carbon_df)
